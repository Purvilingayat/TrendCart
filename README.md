# TrendCart
TrendCart is an e-commerce website developed to provide users with a smooth and user-friendly online shopping experience.

## Features
- Product listing
- Responsive design
- Shopping cart functionality
- User-friendly interface
- Modern UI design

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Objective
The objective of this project is to create a simple and responsive e-commerce platform that allows users to browse products and experience an interactive shopping interface.

## Solo Project
This project was developed as part of an academic project.
